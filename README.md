## ⚙️ Configuration

### Adding Video Sources

1. Go to the "Settings" page
2. Click the "Add" button
3. Fill in the video source information:
   - **ID**: Unique identifier (e.g., mysource)
   - **Name**: Display name (e.g., My Source)
   - **API Address**: Apple CMS API address

### Player Configuration

You can configure a custom player address in the settings page, supporting any player that supports URL parameters.

### CORS Proxy

If you encounter cross-origin issues, you can configure a CORS proxy address in the settings.

### Cache Management

- Enable/disable API caching
- View cache statistics
- Clear cache

## 🔧 API Support

Supports standard Apple CMS API format:

- List API: `?ac=videolist&pg=1`
- Detail API: `?ac=videolist&ids=123`
- Search API: `?ac=videolist&wd=keyword`
